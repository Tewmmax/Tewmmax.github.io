# Tewmmax.github.io# 🐷 Электронная копилка

## Описание проекта
Электронная копилка на базе ESP32 с автоматическим распознаванием монет номиналом 1, 2, 5 и 10 рублей. Устройство отображает текущий баланс на LCD-дисплее и отправляет данные на GitHub для удалённого мониторинга.

## Технические характеристики
- **Микроконтроллер:** ESP32
- **Датчики:** 4 ИК-прерывателя (симуляция через кнопки)
- **Дисплей:** LCD 16x2 I²C
- **Индикация:** Светодиод
- **Связь:** Wi-Fi (отправка данных на GitHub)

## Функциональные возможности
1. Автоматическое распознавание монет 1₽, 2₽, 5₽, 10₽
2. Подсчёт и отображение текущего баланса
3. Сохранение суммы при отключении питания (EEPROM)
4. Сброс баланса (удержание кнопки 3 секунды)
5. Удалённый мониторинг через веб-интерфейс

## Веб-интерфейс
- Просмотр текущего баланса
- История операций (последние 10)
- Автообновление каждые 10 секунд
